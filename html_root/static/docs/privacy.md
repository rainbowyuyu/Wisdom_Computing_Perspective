# 智算视界 (WisComPer) 隐私政策

**生效日期：** 2026年1月16日

智算视界（以下简称“我们”或“本平台”）深知个人信息对您的重要性，并会尽力保护您的个人信息安全可靠。本隐私政策旨在帮助您了解我们如何收集、使用和存储您的个人信息。

## 1. 信息收集

在使用本平台服务时，我们可能会收集以下信息：

*   **账户信息**：当您注册账户时，我们会收集您的用户名和加密后的密码凭证。我们不会明文存储您的密码。
*   **用户上传内容**：您上传至“智能识别”功能的图片文件。
*   **生成内容**：您通过本平台生成的数学公式（LaTeX代码）及动画视频文件。
*   **算式库数据**：您主动保存到“我的算式”中的公式数据及备注信息。
*   **访问日志**：为了维护系统稳定，我们的服务器可能会自动记录您的 IP 地址、浏览器类型、请求时间等基础日志信息。

## 2. 信息使用

我们收集的信息主要用于：

*   提供核心服务功能（如 OCR 识别、动画渲染）。
*   改善用户体验和优化算法模型。
*   保障账户安全，防止欺诈和滥用。

**特别声明：** 您上传的用于识别的图片仅在内存或临时存储中处理，处理完成后会定期自动清除，**我们不会将您的私有图片用于其他商业用途或出售给第三方**。

## 3. 信息存储与保护

*   **数据存储**：您的账户信息和算式库数据存储在安全的 MySQL 数据库中。生成的视频文件存储在受保护的静态资源服务器上。
*   **安全措施**：我们使用行业标准的加密技术（如 Bcrypt）来保护您的密码。我们采取合理的物理、电子和管理措施来防止数据泄露。

## 4. 第三方服务

为了提供部分 AI 能力，我们可能会调用第三方大模型接口（如阿里云 DashScope）。在调用过程中，我们会对数据进行脱敏处理，且仅传输必要的文本或图像特征，不会传输您的个人身份信息。

## 5. 您的权利

您有权：
*   随时查看、修改或删除您在“我的算式”中保存的数据。
*   联系我们注销您的账户（注销后所有数据将被永久删除）。
